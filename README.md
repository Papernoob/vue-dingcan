# vue-dingcan
毕设订餐系统--需要配合自己写的express框架后台
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
express后台开启后
npm run serve

```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
